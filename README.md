# BijliSmart App

BijliSmart is a Flutter-based mobile application designed to monitor and manage electricity usage efficiently. The app focuses on providing users with insights into their energy consumption through a simple and user-friendly interface.

---

## Project Overview

The application helps users track and understand their electricity usage patterns. It is designed with a clean UI and modular architecture to ensure scalability and maintainability.
